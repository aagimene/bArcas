After installation, the program (ProSurf3.exe) will reside in your
[INSTALLDIR] folder or directory.  The default location of the 
[INSTALLDIR] is C:\Program Files\New Wave Systems\ProSurf3.  The
installation also creates the following directories:

DB         location of all of the sample SRF sample files
DB\KAYAK   location of a group of native kayak hull files
AIRFOILS   location of the 1000+ airfoil shape files from UIUC
ARTICLES   location of numerous technical articles or white papers
TUTORIALS  location of various tutorials and examples

You should consider creating a special folder or directory for your 
boat data files so that they don't get mixed in with these files.

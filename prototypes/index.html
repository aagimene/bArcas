bArcas — prototypes

Small, self-contained interactive explorations of single concepts from kayak / hull design. Vanilla HTML + CSS + JS; no build step. github.com/aagimene/bArcas

Available

  • stability/

    Editable 2D hull cross-section with live KB / BM / KM / GM / GZ, primary and secondary stability metrics (peak GZ, AVS, area under GZ, GZ@30°), and an annotated GZ curve. Drag control points to reshape the hull and watch the metacenter, righting arm, and stability curve update in real time.

    Background: wiki/theory/stability.md

  • loft/

    3D kayak hull lofted from a curved spine (rocker) and editable cross-sections, with a rotatable Three.js view, side-view rocker editor, and per-station body-plan editor. Live recompute on every edit.

    Plan: prototypes/loft-plan.md · Currently: Phase A (layout skeleton + placeholder geometry).

Planned

  • drag future

    Frictional drag (ITTC 1957), wave-making (Michell thin-ship), and total resistance vs. speed for a given lofted hull. Reads geometry from the loft prototype.

  • seakeeping future

    Roll period, response in a beam sea, and reserve-buoyancy behaviour at the bow. Strip-theory motion analysis on the lofted hull.

Conventions

  • SI units internally (m, kg, rad, m/s); imperial / degrees only in the UI.
  • Hover any variable name for its definition; click any value for its equation.
  • Mobile-friendly: layouts collapse to a single column on narrow screens.
